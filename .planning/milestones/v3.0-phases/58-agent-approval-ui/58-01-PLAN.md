- `build_toast_text(&self, data: &ProposalToastRenderData, viewport_w: f32, viewport_h: f32) -> Vec<ProposalToastTextLabel>`: Returns ProposalToastTextLabel structs (text, x, y, color) for 2 lines. Line 1: description truncated to 60 chars. Line 2: keyboard hint + remaining seconds.
- Define `ProposalToastTextLabel { pub text: String, pub x: f32, pub y: f32, pub color: Rgb }`.
- Add `#[cfg(test)] mod tests` with tests for rect count, position bounds, text label count, description truncation.

- Define `ProposalOverlayRenderData { proposals: Vec<(String, String)>, selected: usize, diff_preview: String }` where tuples are (description, action).
- Define `ProposalOverlayRenderer { cell_width: f32, cell_height: f32 }`.
- `new(cell_width, cell_height) -> Self`
- `build_overlay_rects(&self, viewport_w: f32, viewport_h: f32, data: &ProposalOverlayRenderData) -> Vec<RectInstance>`: Returns backdrop (full viewport, [0.03, 0.03, 0.03, 0.88]), panel (80% width centered, full height minus 2 * cell_height for tab/status), selected row highlight.
- `build_overlay_text(&self, viewport_w: f32, viewport_h: f32, data: &ProposalOverlayRenderData) -> Vec<ProposalOverlayTextLabel>`: Returns header, proposal descriptions (selected marked with ">"), diff preview lines (max 50), footer hint line with Ctrl+Shift+Y/N/A keys.
