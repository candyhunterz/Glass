1. **TokenBudget enum** with four variants: OneLine (limit 10), Summary (limit 100), Detailed (limit 500), Full (usize::MAX). Add `token_limit(self) -> usize` method. Derive Debug, Clone, Copy, PartialEq, Eq, Serialize.

2. **CompressedOutput struct** with fields: budget (TokenBudget), text (String), record_ids (Vec<i64>), token_count (usize), truncated (bool). Derive Debug, Clone, Serialize.

3. **compress() function** signature: `pub fn compress(records: &[crate::soi::OutputRecordRow], summary: &crate::soi::CommandOutputSummaryRow, budget: TokenBudget) -> CompressedOutput`

Implementation logic:
- **OneLine**: Count Error-severity records. If any exist, find first Error record (min id), extract its file_path, format as "{N} error(s) in {file_path}". If no file_path or no errors, fall back to `summary.one_line.clone()`. Token count via `split_whitespace().count()`.
- **Full**: Bypass budget loop. Build multi-line text with all records (format each record as "{record_type}: {severity} {file_path} - {message_from_data}"). Set truncated = false, record_ids = all record ids.
- **Summary/Detailed**: Sort records by severity_rank ASC (Error=0, Warning=1, Info=2, Success=3, None=4), then by id ASC. Greedily include records until token count would exceed budget.token_limit(). For each included record, format a one-line entry and add its id to record_ids. Set truncated = true if any records were excluded.

4. **Helper functions**:
- `severity_rank(severity: Option<&str>) -> u8` -- returns 0 for Error, 1 for Warning, 2 for Info, 3 for Success, 4 for None
- `format_record(row: &OutputRecordRow) -> String` -- extracts a human-readable one-liner from the record. Parse `row.data` as `serde_json::Value` to extract message/name/package field based on `row.record_type`. Use record_type + severity + file_path + message_prefix. If JSON parse fails, use "{record_type} (details unavailable)".
- `estimate_tokens(text: &str) -> usize` -- `text.split_whitespace().count()` per project convention.

```rust
/// Compress output records for a command at the given token budget level.
///
/// Fetches the summary and records from the DB, then runs the compression
/// engine. Returns None if the command has no SOI data.
pub fn compress_output(
    &self,
    command_id: i64,
    budget: crate::compress::TokenBudget,
) -> Result<Option<crate::compress::CompressedOutput>> {
    let summary = match crate::soi::get_output_summary(&self.conn, command_id)? {
        Some(s) => s,
        None => return Ok(None),
    };
    let records = crate::soi::get_output_records(&self.conn, command_id, None, None, None, 10000)?;
    Ok(Some(crate::compress::compress(&records, &summary, budget)))
}
```
