Create `crates/glass_agent/src/worktree_manager.rs`:

1. `WorktreeManager` struct:
   - `base_dir: PathBuf` (defaults to `~/.glass/worktrees/`)
   - `db: WorktreeDb`

2. `WorktreeManager::new(base_dir: PathBuf, db: WorktreeDb) -> Self`

3. `WorktreeManager::new_default() -> Result<Self>` -- uses `dirs::home_dir()/.glass/worktrees/` and `WorktreeDb::open_default()`

4. `create_worktree(&self, project_root: &Path, proposal_id: &str, file_changes: &[(String, String)]) -> Result<WorktreeHandle>`:
   - Generate UUID for id
   - Compute worktree_path = base_dir/id
   - **REGISTER in DB BEFORE creating** (crash safety invariant from STATE.md)
   - Detect git vs non-git: `git2::Repository::discover(project_root)`
   - Git path: `repo.worktree(&id, &worktree_path, None)?` then write file_changes into worktree
   - Non-git path: `create_dir_all(&worktree_path)` then write file_changes into worktree
   - On error: delete pending row, propagate error
   - Return WorktreeHandle with changed_files = file paths from file_changes

5. `generate_diff(&self, handle: &WorktreeHandle) -> Result<String>`:
   - For each changed_file: read original from project_root (empty string if doesn't exist = new file), read modified from worktree_path
   - Handle binary: if `read_to_string` fails, emit `"Binary file {path} changed\n"` placeholder
   - Call `diffy::create_patch(&original, &modified)` and format with `--- a/{path}` / `+++ b/{path}` header
   - Concatenate all patches into single string

6. `apply(&self, handle: WorktreeHandle) -> Result<()>`:
   - For each changed_file: `fs::create_dir_all(parent)`, `fs::copy(worktree/rel, project/rel)`
   - Call `self.cleanup(handle)`

7. `dismiss(&self, handle: WorktreeHandle) -> Result<()>`:
   - Just call `self.cleanup(handle)` (no file copies)

8. Private `cleanup(&self, handle: WorktreeHandle) -> Result<()>`:
   - Match on handle.kind:
     - Git: open repo, find_worktree by id, prune with `WorktreePruneOptions::valid(true)`, then `remove_dir_all` as belt-and-suspenders
     - TempDir: just `remove_dir_all`
   - Delete pending_worktrees row via `self.db.delete_pending_worktree(&handle.id)`

9. `prune_orphans(&self) -> Result<()>`:
   - List all pending_worktrees from DB
   - For each: if worktree_path exists, try git prune (discover repo, find worktree, prune), then `remove_dir_all`
   - Delete the DB row regardless

Write tests in `#[cfg(test)] mod tests` using `tempfile::TempDir` for both git and non-git scenarios. For git tests: `git2::Repository::init(temp_dir)` to create a test repo.

IMPORTANT: Do NOT use `head` command in any test or verify step (not available in this environment).
IMPORTANT: Use `WorktreePruneOptions::new()` then `.valid(true)` -- this forces prune even when the worktree path still exists on disk.
IMPORTANT: The worktree name (first arg to `repo.worktree()`) MUST NOT contain slashes -- use the UUID string directly.
