```rust
#[derive(Debug, Deserialize, schemars::JsonSchema)]
pub struct RegisterParams {
    #[schemars(description = "Human-readable agent name")]
    pub name: String,
    #[schemars(description = "Agent type (e.g. 'claude-code', 'cursor', 'human')")]
    pub agent_type: String,
    #[schemars(description = "Project root path for scoping")]
    pub project: String,
    #[schemars(description = "Current working directory")]
    pub cwd: String,
    #[schemars(description = "OS process ID for liveness detection")]
    pub pid: Option<u32>,
}

#[derive(Debug, Deserialize, schemars::JsonSchema)]
pub struct DeregisterParams {
    #[schemars(description = "Agent UUID to deregister")]
    pub agent_id: String,
}

#[derive(Debug, Deserialize, schemars::JsonSchema)]
pub struct ListAgentsParams {
    #[schemars(description = "Project root path to list agents for")]
    pub project: String,
}

#[derive(Debug, Deserialize, schemars::JsonSchema)]
pub struct StatusParams {
    #[schemars(description = "Agent UUID")]
    pub agent_id: String,
    #[schemars(description = "New status (e.g. 'active', 'idle', 'editing')")]
    pub status: String,
    #[schemars(description = "Current task description")]
    pub task: Option<String>,
}

#[derive(Debug, Deserialize, schemars::JsonSchema)]
pub struct HeartbeatParams {
    #[schemars(description = "Agent UUID to refresh heartbeat for")]
    pub agent_id: String,
}
```

**Tool handlers** (add inside `#[tool_router] impl GlassServer` block, after glass_pipe_inspect):

1. `glass_agent_register` -- opens CoordinationDb via `self.coord_db_path.clone()` in `spawn_blocking`, calls `db.register(...)`, then `db.list_agents(&params.project)` to get count, returns JSON `{ agent_id, agents_active }`.

2. `glass_agent_deregister` -- opens DB, calls `db.deregister(&params.agent_id)`, returns JSON `{ ok: result }`.

3. `glass_agent_list` -- opens DB, calls `db.prune_stale(600)` (10min timeout per Phase 31 success criteria), then `db.list_agents(&params.project)`, returns JSON `{ agents: [...] }`. Each agent serialized via serde (AgentInfo derives Serialize).

4. `glass_agent_status` -- opens DB, calls `db.update_status(&params.agent_id, &params.status, params.task.as_deref())`, returns JSON `{ ok: result }`.

5. `glass_agent_heartbeat` -- opens DB, calls `db.heartbeat(&params.agent_id)`, returns JSON `{ ok: result }`.

All handlers follow the exact spawn_blocking + map_err(internal_err) + Content::json pattern from existing tools.
