## Tech Debt

### Phase 5: History Database Foundation
- **Test isolation:** `test_resolve_db_path_global_fallback` fails on machines with `~/.glass/` directory (needs environment variable override or home_dir parameter)

### Phase 6: Output Capture + Writer Integration
- **Empty command text:** `insert_command` stores empty string for command text (metadata + output captured correctly; command text extraction from terminal grid was deferred)
- **No throughput benchmark:** Architecture is non-blocking (OutputBuffer in PTY thread, DB writes on main thread via AppEvent) but no quantitative benchmark exists

### Cross-Phase
- **HIST-05 no runtime trigger:** `HistoryDb::prune()` exists and is tested but nothing calls it during normal operation. Database will grow without bounds until manual pruning or a future scheduled trigger is added.
