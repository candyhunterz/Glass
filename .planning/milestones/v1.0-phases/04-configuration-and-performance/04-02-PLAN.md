<task type="auto">
  <name>Task 1: Add performance instrumentation</name>
  <files>src/main.rs, Cargo.toml</files>
  <action>
1. Add `memory-stats = "1.2"` to workspace Cargo.toml [workspace.dependencies]. Add `memory-stats.workspace = true` to root Cargo.toml [dependencies].

2. In src/main.rs, add cold start timing:
   - At the very top of `main()`, before UTF-8 setup: `let cold_start = std::time::Instant::now();`
   - Add `cold_start: std::time::Instant` field to `Processor` struct
   - Pass `cold_start` into Processor initialization
   - Add `first_frame_logged: bool` field to `WindowContext` (initialized to `false`)
   - In the `RedrawRequested` handler, after the first successful `draw_frame()` call, if `!ctx.first_frame_logged`: log `tracing::info!("PERF cold_start={:?}", self.cold_start.elapsed());` and set `first_frame_logged = true`

3. Add input latency measurement:
   - In the `KeyboardInput` handler for `ElementState::Pressed`, immediately capture `let key_start = std::time::Instant::now();`
   - After the `pty_sender.send(PtyMsg::Input(bytes))` call, log `tracing::trace!("PERF key_latency={:?}", key_start.elapsed());`
   - Use `trace!` level so it only appears when RUST_LOG=trace (doesn't spam normal usage)

4. Add idle memory measurement:
   - After the cold start log line (step 2), also log memory: `if let Some(usage) = memory_stats::memory_stats() { tracing::info!("PERF memory_physical={:.1}MB", usage.physical_mem as f64 / 1_048_576.0); }`
   - This gives a startup memory snapshot; idle memory can be checked later via Task Manager

5. Important: All PERF log lines use a consistent `PERF` prefix for easy grep/filtering.

Anti-patterns to avoid:
- Do NOT measure with debug builds (results meaningless for PERF targets)
- Do NOT add sleep or artificial delays for measurement
- Do NOT add memory-stats behind a feature flag (it's tiny, always useful)
