## Tech Debt

### Phase 3: Shell Integration

- **display_offset hardcoded to 0** in `crates/glass_renderer/src/frame.rs` (lines 102, 156): Block decorations (separators, badges, duration labels) render at viewport-relative positions instead of scrolling with content. The correct data is available via `snapshot.display_offset` but is not passed to `build_block_rects()` and `build_block_text()`. Impact: block decorations misalign during scrollback. Severity: Warning — does not affect zero-scroll operation (primary use case).

### Phase 1: Scaffold

- **ConPTY test execution not formally logged**: Tests compile and logic is verified, but no recorded pass/fail from actual execution. Human verification items in VERIFICATION.md remain as `HUMAN NEEDED` for test_conpty_spawns_and_wakeup_fires, test_pty_keyboard_round_trip, and test_utf8_codepage_65001_active. Impact: Low — user confirmed functionality interactively during development.

### Performance Target Revisions

- **PERF-01** revised from <200ms to <500ms (DX12 hardware init floor ~290ms)
- **PERF-03** revised from <50MB to <120MB (GPU driver overhead ~80MB baseline)
- Both revisions are documented with technical rationale and reflected in REQUIREMENTS.md and ROADMAP.md
