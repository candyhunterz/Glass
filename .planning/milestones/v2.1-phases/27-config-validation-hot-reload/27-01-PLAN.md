    ```rust
    #[derive(Debug, Clone)]
    pub struct ConfigError {
        pub message: String,
        pub line: Option<usize>,    // 1-based
        pub column: Option<usize>,  // 1-based
        pub snippet: Option<String>, // the offending line from TOML source
    }
    ```

    3. Implement `std::fmt::Display` for ConfigError:

       - Convert byte offset to line/col by counting newlines in `s[..span.start]`
       - Extract the offending line via `s.lines().nth(line - 1)`
       - Use `e.message()` for the error message string

       - Returns `self.font_family != other.font_family || self.font_size != other.font_size`
       - Note: f32 direct comparison is fine here since these are parsed from TOML, not computed.

    8. Keep existing `load()` and `load_from_str()` unchanged (backward compatible). `load()` can optionally be updated to use `load_validated()` internally and log the structured error, but must still return `Self` (not Result).
