Create all content pages. Each page should be practical and concise -- document actual shipped behavior (all 29 prior phases are complete).

**Installation pages:**

`installation/windows.md`:
- MSI installer: download from GitHub Releases, double-click to install to Program Files, adds `glass` to PATH
- Winget: `winget install Glass.Terminal` (once published)
- Note: unsigned binary may trigger SmartScreen; click "More info" then "Run anyway"

`installation/macos.md`:
- DMG: download from GitHub Releases, drag Glass.app to Applications
- Homebrew: `brew install <GITHUB_USER>/glass/glass` (once tap published)
- Gatekeeper workaround: `xattr -cr /Applications/Glass.app` or right-click > Open
- Minimum macOS version: 11.0 (Big Sur)

`installation/linux.md`:
- deb package: download from GitHub Releases, `sudo dpkg -i glass_*.deb`
- Dependencies: GPU drivers (Vulkan or OpenGL), standard terminal libraries

**Getting started:**

`getting-started.md`:
- First launch: Glass opens with default shell (auto-detected)
- Shell integration: explain that Glass automatically detects command boundaries
- Basic navigation: scroll through command blocks, each block shows exit code, duration, working directory
- Key shortcuts table: Ctrl+Shift+T (new tab), Ctrl+Shift+D (split pane), Ctrl+Shift+F (search), Ctrl+Shift+Z (undo), Ctrl+Shift+U (check for updates)
- Config file location: `~/.glass/config.toml`

**Feature pages:**

`features/blocks.md`: Command blocks concept -- each command rendered as a distinct block with exit code badge (green checkmark for 0, red X for non-zero), duration, working directory. Explain how Glass understands command structure via shell integration.

`features/search.md`: Ctrl+Shift+F opens search overlay. FTS5-powered search across all terminal history stored in SQLite. Describe the search UI, result navigation, and that search persists across sessions.

`features/undo.md`: Ctrl+Shift+Z triggers file undo. Explain: Glass snapshots files before commands modify them, allowing restoration. Cover the undo notification, which files are tracked, snapshot retention settings (max_count, max_size_mb, retention_days in [snapshot] config section).

`features/pipes.md`: Pipeline inspection -- Glass captures intermediate stage output for piped commands. Explain how to view pipe stages, the auto-expand on failure behavior, and [pipes] config options (enabled, max_capture_mb, auto_expand).

`features/tabs-panes.md`: Ctrl+Shift+T for new tab, Ctrl+Shift+D for vertical split, Ctrl+Shift+W to close. Tab bar shows tab names. Pane focus with mouse click. Each pane runs its own shell session.

`features/history.md`: SQLite-backed command history with FTS5 full-text search. CLI query interface via `glass history` commands. History database stored alongside config. [history] config section (max_output_capture_kb).

**Reference pages:**

`configuration.md`: Full config.toml reference with ALL options and defaults. Document each section:
- Top-level: font_family (platform defaults: Consolas/Menlo/Monospace), font_size (14.0), shell (auto-detected)
- [history]: max_output_capture_kb (50)
- [snapshot]: enabled (true), max_count (1000), max_size_mb (500), retention_days (30)
- [pipes]: enabled (true), max_capture_mb (10), auto_expand (true)
- Include a complete example config.toml
- Note: config changes are applied live via hot-reload (no restart needed)

`mcp-server.md`: MCP server setup for AI assistant integration. Explain what MCP is, how to configure Claude Desktop or other MCP clients to connect to Glass's MCP server. Document available MCP tools/resources that Glass exposes (command history queries, undo operations, etc.).

`troubleshooting.md`: Common issues:
- macOS Gatekeeper blocks launch (xattr workaround)
- GPU driver issues (Vulkan/OpenGL requirements)
- Shell not detected (manual shell config)
- Config parse errors (error overlay explains what went wrong)
- Font not found (fallback behavior)
- Auto-update issues

Read `crates/glass_core/src/config.rs` for exact config field names and defaults before writing configuration.md.
