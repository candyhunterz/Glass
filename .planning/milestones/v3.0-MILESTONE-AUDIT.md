**Root cause:** Writer thread (`glass-agent-writer`) not signaled on budget exhaustion.

- **AGTR-06 (partial):** `BudgetTracker.is_exceeded()` correctly pauses proposals but writer thread continues forwarding activity events to claude stdin

**Impact:** Low — proposals ARE correctly paused. The subprocess receives events it cannot act on until restart. This is a resource efficiency gap, not a functional failure.
