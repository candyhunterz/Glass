<objective>
Gate all pipe capture behavior behind pipes.enabled config flag.

Purpose: Currently pipes.enabled=false only skips temp file reading in main.rs. Shell scripts still rewrite pipelines, BlockManager still accumulates empty CapturedStage entries, and empty PipeStageRow records could get persisted. This plan makes the disable truly comprehensive.

Output: Three-layer gating: PTY env var -> shell script early return -> main.rs event skip
</objective>
