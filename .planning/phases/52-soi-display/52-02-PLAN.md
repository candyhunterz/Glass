**Block field population (SOID-01):**
After storing `session.last_soi_summary`, check if SOI display is enabled:
```rust
let soi_enabled = self.config.soi.as_ref().map(|s| s.enabled).unwrap_or(true);
```
If `soi_enabled`, find the last `Complete` block and set its SOI fields:
```rust
if soi_enabled {
    if let Some(block) = session.block_manager.blocks_mut()
        .iter_mut()
        .rev()
        .find(|b| b.state == glass_terminal::BlockState::Complete)
    {
        block.soi_summary = Some(summary.clone());
        block.soi_severity = Some(severity.clone());
    }
}
```
Use `rev().find(|b| b.state == BlockState::Complete)` -- NOT `current_block_mut()` -- to handle the race condition where the next PromptStart may have already advanced the current block pointer.

**Shell hint line injection (SOID-02):**
After block population, check config for shell_summary:
```rust
let shell_summary_enabled = self.config.soi.as_ref()
    .map(|s| s.enabled && s.shell_summary)
    .unwrap_or(false);
let min_lines = self.config.soi.as_ref()
    .map(|s| s.min_lines)
    .unwrap_or(0);
```
If enabled and summary is non-empty and raw_line_count >= min_lines (or min_lines == 0):
```rust
if shell_summary_enabled && !summary.is_empty()
    && (min_lines == 0 || raw_line_count >= min_lines as i64)
{
    let hint = format!("\x1b[2m[glass-soi] {}\x1b[0m\r\n", summary);
    let _ = session.pty_sender.send(
        glass_terminal::PtyMsg::Input(std::borrow::Cow::Owned(hint.into_bytes()))
    );
}
```

Do NOT include any OSC sequences (`\x1b]`) in the hint line -- only SGR (`\x1b[`) for dim styling. This avoids triggering OscScanner and causing shell integration side effects.

**Important:** The summary and severity variables are moved into `last_soi_summary` first. Clone them before the move, or restructure to clone before storing in SoiSummary. The simplest approach: clone summary and severity before storing in last_soi_summary, then use the clones for block fields and hint line.

Full handler structure:
```rust
AppEvent::SoiReady { window_id, session_id, command_id, summary, severity, raw_line_count } => {
    if let Some(ctx) = self.windows.get_mut(&window_id) {
        if let Some(session) = ctx.session_mux.session_mut(session_id) {
            if session.last_command_id == Some(command_id) {
                // Store session-level summary
                session.last_soi_summary = Some(glass_mux::session::SoiSummary {
                    command_id,
                    one_line: summary.clone(),
                    severity: severity.clone(),
                });

                tracing::debug!("SOI ready for cmd {}: {}", command_id, summary);

                // SOID-01: Populate block fields if enabled
                let soi_enabled = self.config.soi.as_ref().map(|s| s.enabled).unwrap_or(true);
                if soi_enabled {
                    if let Some(block) = session.block_manager.blocks_mut()
                        .iter_mut()
                        .rev()
                        .find(|b| b.state == glass_terminal::BlockState::Complete)
                    {
                        block.soi_summary = Some(summary.clone());
                        block.soi_severity = Some(severity.clone());
                    }
                }

                // SOID-02: Inject hint line if shell_summary enabled
                let shell_summary_enabled = self.config.soi.as_ref()
                    .map(|s| s.enabled && s.shell_summary)
                    .unwrap_or(false);
                let min_lines = self.config.soi.as_ref()
                    .map(|s| s.min_lines)
                    .unwrap_or(0);
                if shell_summary_enabled && !summary.is_empty()
                    && (min_lines == 0 || raw_line_count >= min_lines as i64)
                {
                    let hint = format!("\x1b[2m[glass-soi] {}\x1b[0m\r\n", summary);
                    let _ = session.pty_sender.send(
                        glass_terminal::PtyMsg::Input(
                            std::borrow::Cow::Owned(hint.into_bytes())
                        )
                    );
                }
            }
        }
        ctx.window.request_redraw();
    }
}
```
