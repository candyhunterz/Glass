GREEN phase: Implement `pub fn build_decoration_rects(&self, snapshot: &GridSnapshot) -> Vec<RectInstance>` on GridRenderer. The method:
1. Creates an empty Vec<RectInstance>
2. Gets `line_offset` from `snapshot.display_offset as i32`
3. Iterates `snapshot.cells`
4. Skips cells where flags intersect `WIDE_CHAR_SPACER | LEADING_WIDE_CHAR_SPACER`
5. For cells with UNDERLINE or STRIKEOUT flags:
   - Computes `rect_width = if is_wide { cell_width * 2.0 } else { cell_width }`
   - Computes `x = cell.point.column.0 as f32 * self.cell_width`

   - Gets color via `rgb_to_color(cell.fg, 1.0)` (same helper used by build_rects)
   - For UNDERLINE: pushes RectInstance with pos `[x, y + self.cell_height - 1.0, rect_width, 1.0]`
   - For STRIKEOUT: pushes RectInstance with pos `[x, y + (self.cell_height / 2.0).floor(), rect_width, 1.0]`
6. Returns the vec
