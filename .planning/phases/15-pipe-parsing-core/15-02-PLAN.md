  <action>
Update crates/glass_pipes/src/types.rs to replace the stub StageBuffer implementation with the full version:

1. Add a private helper `fn is_binary_data(data: &[u8]) -> bool`:
   - Adapted from glass_history::output::is_binary() pattern
   - Sample first 8192 bytes (or all data if shorter)
   - Count bytes that are NOT valid text (not in 0x09, 0x0A, 0x0D, 0x20..0x7E range, and not valid UTF-8 continuation bytes 0x80..0xBF when following a valid UTF-8 lead byte)
   - Simpler approach matching existing codebase: count bytes < 0x08 or bytes in 0x0E..0x1F (control chars excluding tab, newline, carriage return) as non-text. If >30% of sampled bytes are non-text, it's binary.
   - Return true if non-text ratio exceeds 30%

2. Implement `StageBuffer::append(&mut self, data: &[u8])`:
   - Track total_bytes += data.len()
   - If not in overflow mode: extend head with data. If head.len() > policy.max_bytes, transition to overflow:
     - Set overflow = true
     - Truncate head to policy.sample_size
     - Start tail with the last policy.sample_size bytes of the data that caused overflow
   - If in overflow mode: extend tail with data. If tail.len() > policy.sample_size, drain the excess from the front (rolling window keeps latest bytes)

3. Implement `StageBuffer::finalize(self) -> FinalizedBuffer`:
   - Check binary: call is_binary_data on &self.head (first bytes of data)
   - If binary: return FinalizedBuffer::Binary { size: self.total_bytes }
   - If overflow: return FinalizedBuffer::Sampled { head, tail, total_bytes }
   - Otherwise: return FinalizedBuffer::Complete(self.head)

4. Add `StageBuffer::total_bytes(&self) -> usize` getter.
