<task type="auto" tdd="true">
  <name>Task 1: OscScanner with TDD</name>
  <files>crates/glass_terminal/src/osc_scanner.rs, Cargo.toml, crates/glass_terminal/Cargo.toml</files>
  <behavior>
    - Test: scan(b"\x1b]133;A\x07") produces OscEvent::PromptStart
    - Test: scan(b"\x1b]133;B\x07") produces OscEvent::CommandStart
    - Test: scan(b"\x1b]133;C\x07") produces OscEvent::CommandExecuted
    - Test: scan(b"\x1b]133;D;0\x07") produces OscEvent::CommandFinished { exit_code: Some(0) }
    - Test: scan(b"\x1b]133;D;1\x07") produces OscEvent::CommandFinished { exit_code: Some(1) }
    - Test: scan(b"\x1b]133;D\x07") produces OscEvent::CommandFinished { exit_code: None }
    - Test: scan(b"\x1b]7;file://HOST/C:/Users/foo\x07") produces OscEvent::CurrentDirectory("C:/Users/foo")
    - Test: scan(b"\x1b]7;file://HOST/C:/Users/foo\x1b\\") produces OscEvent::CurrentDirectory (ST terminator)
    - Test: split buffer -- scan(b"\x1b]133;") then scan(b"A\x07") produces PromptStart
    - Test: split buffer mid-OSC -- scan(b"\x1b]") then scan(b"133;C\x07") produces CommandExecuted
    - Test: non-OSC data passes through without events
    - Test: interleaved normal bytes and OSC sequences produce correct events
    - Test: OSC 9;9 (ConEmu CWD) produces OscEvent::CurrentDirectory
  </behavior>
  <action>
    Add `url = "2"` to workspace Cargo.toml and `url.workspace = true` to glass_terminal/Cargo.toml.

    Create osc_scanner.rs with:

    1. `OscEvent` enum: PromptStart, CommandStart, CommandExecuted, CommandFinished { exit_code: Option<i32> }, CurrentDirectory(String)
    2. `ScanState` enum: Ground, Escape (saw \x1b), OscStart (saw \x1b]), Accumulating (reading payload)
    3. `OscScanner` struct with state: ScanState, buffer: Vec<u8>
    4. `scan(&mut self, data: &[u8]) -> Vec<OscEvent>` method:
       - Iterate byte by byte through data
       - In Ground: watch for \x1b -> transition to Escape
       - In Escape: if ] -> OscStart, else -> Ground
       - In OscStart: begin Accumulating, push byte to buffer
       - In Accumulating: push bytes until \x07 (BEL) or \x1b\\ (ST) terminators
       - On terminator: parse buffer contents, emit OscEvent if recognized, clear buffer
       - Buffer parsing: split on ';', check first segment for "133" or "7" or "9;9"
       - For 133: match second segment A/B/C/D, extract exit code from D's optional third segment
       - For 7: parse file:// URL using `url` crate, extract path (handle Windows /C:/ prefix)
       - For 9;9: extract CWD path directly (ConEmu format, no URL encoding)

    Follow RED-GREEN-REFACTOR: write all tests first (they fail), then implement scan() to make them pass.
  </action>
  <verify>
    <automated>cargo test -p glass_terminal osc_scanner -- --nocapture</automated>
  </verify>
  <done>All OscScanner tests pass. Scanner handles OSC 133 A/B/C/D, OSC 7 file:// URLs, OSC 9;9, both BEL and ST terminators, and split-buffer edge cases.</done>
