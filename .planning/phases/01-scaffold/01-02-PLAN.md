Implement three files following the exact patterns from RESEARCH.md:

1. **crates/glass_renderer/src/surface.rs** — GlassRenderer struct (RESEARCH.md "Pattern 2"):
   - Struct fields: `device: wgpu::Device`, `queue: wgpu::Queue`, `surface: wgpu::Surface<'static>`, `surface_config: wgpu::SurfaceConfiguration`
   - `pub async fn new(window: Arc<winit::window::Window>) -> Self`:
     - Create `wgpu::Instance::default()` (auto-selects DX12 on Windows)
     - `instance.create_surface(window.clone())` for surface
     - Request adapter with `PowerPreference::HighPerformance` and `compatible_surface: Some(&surface)`
     - Log backend via `tracing::info!("GPU backend: {:?}", adapter.get_info().backend)`
     - Request device with `DeviceDescriptor::default()`
     - Configure surface: format from `caps.formats[0]`, `PresentMode::AutoVsync`, `desired_maximum_frame_latency: 2`
   - `pub fn draw(&mut self)`:
     - `surface.get_current_texture()` with match on errors:
       - `SurfaceError::Outdated | SurfaceError::Lost` -> reconfigure surface and return (DO NOT PANIC — Pitfall 3)
       - Other errors -> log and return
     - Create render pass clearing to dark gray: `Color { r: 0.1, g: 0.1, b: 0.1, a: 1.0 }`
     - Submit command encoder and present frame
   - `pub fn resize(&mut self, width: u32, height: u32)`:
     - Guard: return early if width == 0 or height == 0
     - Update surface_config width/height and reconfigure surface

2. **crates/glass_renderer/src/lib.rs** — Public module and re-export:
   ```rust
   pub mod surface;
   pub use surface::GlassRenderer;
   ```

3. **src/main.rs** — Full winit ApplicationHandler (RESEARCH.md "Pattern 1"):
   - Define `Processor` struct with `windows: HashMap<WindowId, WindowContext>`
   - `WindowContext` holds: `window: Arc<Window>`, `renderer: GlassRenderer`
   - `impl ApplicationHandler<AppEvent> for Processor`:
     - `can_create_surfaces()` (REQUIRED — not resumed()! See Pitfall 1):
       - Create window with `event_loop.create_window(Window::default_attributes().with_title("Glass"))`
       - Wrap in Arc
       - `pollster::block_on(GlassRenderer::new(Arc::clone(&window)))` for renderer
       - Insert into windows HashMap
     - `window_event()`:
       - `CloseRequested` -> `event_loop.exit()`
       - `RedrawRequested` -> `ctx.renderer.draw()`
       - `Resized(size)` -> `ctx.renderer.resize(size.width, size.height)`
       - Other events -> ignore for now
     - `user_event()`: match on AppEvent variants (TerminalDirty -> request_redraw, others -> ignore)
     - `about_to_wait()`: no-op for now
   - `fn main()`:
     - Set UTF-8 code page FIRST (Pitfall 5): call `SetConsoleCP(65001)` and `SetConsoleOutputCP(65001)` using `windows_sys::Win32::System::Console` behind `#[cfg(target_os = "windows")]`
     - Init tracing subscriber with env filter
     - Create `EventLoop::<AppEvent>::with_user_event().build().unwrap()`
     - Create Processor with empty HashMap
     - `event_loop.run_app(&mut processor).unwrap()`

   CRITICAL: Add `windows-sys` dependency. In the root Cargo.toml `[dependencies]` section, add:
   `windows-sys = { version = "0.59", features = ["Win32_System_Console"] }`
   Also add it to `[workspace.dependencies]`:
   `windows-sys = { version = "0.59", features = ["Win32_System_Console"] }`

   Do NOT use `WindowBuilder` (deprecated in winit 0.30). Use `Window::default_attributes()`.
   Do NOT use `EventLoop::run()` with a closure (deprecated). Use `EventLoop::run_app()`.
   Do NOT create the window in `resumed()`. Use `can_create_surfaces()`.
