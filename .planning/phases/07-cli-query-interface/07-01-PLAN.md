   - `QueryFilter` struct with fields: `text: Option<String>`, `exit_code: Option<i32>`, `after: Option<i64>`, `before: Option<i64>`, `cwd: Option<String>`, `limit: usize` (default 25). Implement `Default` and `QueryFilter::new()`.

   - `pub fn parse_time(input: &str) -> Result<i64>`: Parse relative time strings ("30m", "1h", "2d") by stripping the suffix, parsing the number, subtracting chrono::Duration from Utc::now().
