Create `packaging/homebrew/glass.rb` with a Homebrew cask formula.

Use `<GITHUB_USER>` as placeholder for GitHub username. Use version `0.1.0`.

The DMG filename pattern is `Glass-#{version}-aarch64.dmg` (from build-dmg.sh).

Formula content:
```ruby
cask "glass" do
  version "0.1.0"
  sha256 "<SHA256>"

  url "https://github.com/<GITHUB_USER>/glass/releases/download/v#{version}/Glass-#{version}-aarch64.dmg",
      verified: "github.com/<GITHUB_USER>/glass/"
  name "Glass Terminal"
  desc "GPU-accelerated terminal emulator with command structure awareness"
  homepage "https://github.com/<GITHUB_USER>/glass"

  livecheck do
    url :url
    strategy :github_latest
  end

  depends_on macos: ">= :big_sur"

  app "Glass.app"

  zap trash: [
    "~/.glass",
  ]
end
```

Add a comment block at the top of the file explaining:
- This formula is for a custom Homebrew tap (not the official homebrew-cask repo)
- To publish: create a GitHub repo named `homebrew-glass`, put this file in `Casks/glass.rb`
- Users install via: `brew tap <GITHUB_USER>/glass && brew install --cask glass`
- SHA256 must be updated from the actual DMG for each release: `shasum -a 256 Glass-*.dmg`
- The official homebrew-cask repo requires notarization (deferred -- PKG-F04)
