### FinalizedBuffer Variant Propagation

All three variants (Complete, Sampled, Binary) correctly handled at every phase boundary: Phase 15 creation -> Phase 16/17 Block storage -> Phase 17 rendering -> Phase 18 DB persistence -> Phase 19 MCP inspection. Verified by integration checker and `test_pipe_stage_buffer_variants`.

### Integration Gaps

**1. CONF-01 (Medium): pipes.enabled does not fully gate capture**

To fully gate: (a) set `GLASS_PIPES_DISABLED=1` env var for shell scripts to check before rewriting, (b) skip PipelineStart/PipelineStage in BlockManager or main.rs when disabled, (c) skip insert_pipe_stages when disabled.

**2. PIPE-02 (Low): classify_pipeline/has_opt_out orphaned in Rust**

`classify_pipeline()` and `has_opt_out()` are exported from `glass_pipes` but never imported by any consuming crate. Shell scripts implement their own opt-out logic. The Rust functions exist as tested library code but have no runtime consumer.
